# DateTime

Display date time in locale format

## Props

| Name        | Default |     Type |   Description |
| ----------- | :-----: | -------: | ------------: |
| `className` |   ``    | {string} |               |
| `date`      |         | {string} | value of date |

## Sample

```jsx
import DateTime from 'components/DateTime';

<DateTime date="2018-02-02" />
```
